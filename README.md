### A recreation of Google Chrome's offline Dino Game built from scratch using JavaScript and Object-Oriented Programming concepts.

## About

This project was created to practice object-oriented design in JavaScript by modeling game entities as independent objects.

## Concepts explored

- Object-oriented programming
- Encapsulation
- Game loop
- Collision detection
- State management
- DOM manipulation

## Technologies

- JavaScript
- HTML5
- CSS3
